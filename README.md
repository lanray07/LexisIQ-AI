# LexisIQ AI

Premium SwiftUI iOS application scaffold for an enterprise legal productivity platform positioned as "The Bloomberg Terminal for Legal Professionals."

## Build

Open `LexisIQAI.xcodeproj` in Xcode 16 or later and run the `LexisIQAI` scheme on an iOS 17+ simulator or device.

Mock AI is enabled by default through `MockAIService`. The remote placeholder endpoint is:

```http
POST https://YOUR_BACKEND_URL.com/lexisiq-ai
```

## Included

- SwiftUI NavigationStack and tabbed shell
- MVVM workflow view model
- SwiftData persistence models
- StoreKit 2 subscription scaffolding
- Speech-to-text and voice recording architecture
- Premium waveform animation
- OCR placeholder architecture
- Knowledge graph visualization
- Swift Charts analytics
- Native PDF generation and share sheet
- Document upload architecture
- WidgetKit and Apple Watch placeholder folders
- Mandatory legal disclaimers on AI output surfaces
